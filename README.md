## 概要
このアプリはズボラな人でも予定を管理できるようにすることを目的としたスマホアプリです。  
主に以下のことができます。

- ライブ壁紙を使って、ホーム画面からタスクの確認ができる
- 音声入力でスクの追加ができる
- チェックリストで完了したタスクを簡単に確認できる
- カレンダーで日付ごとのタスクを確認できる
